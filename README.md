# globalgamejam2026
Puzzle platformer made for Global Game Jam 2026 at TLÜ 30.01.26-01.02.26 with a theme "Mask"

## Built with
- C++23
- Make
- Raylib

## Tools used
- [jsfxr](https://sfxr.me/) for sound effects creation
- [Kenney free assets](https://kenney.nl/assets/1-bit-platformer-pack) for 1-bit platformer pack
- [Soundtrap](https://www.soundtrap.com/musicmakers) for music creation
- [YouTubeToMP3](https://v1.y2mate.nu/) for music sample:[Siidisõsarõ, kadrilaul](https://www.youtube.com/watch?v=3mlT6y1OID0)
